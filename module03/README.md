# Module 03 — File di Consegna

Exercises Ex00–Ex07 from the AI course project.
Focus: prompt design, Excel formulas, data handling, normalization (Min-Max),
and basic visualization.
## Skills demonstrated
- Requirements analysis
- Prompt engineering for structured tasks
- Spreadsheet-based data modeling
- Conditional logic and rule-based computation
- Data normalization and comparison
- Clear separation between input data, logic, and output

## Artifacts

- `Module03.xlsx` — Spreadsheet completo con tutti gli esercizi (Ex01–Ex07)
- `Module03.txt` — Prompt originali e task instructions forniti all’AI
- `export/module03.pdf` — Versione PDF del file di consegna
- `assets/ex06-bar-chart.png` — Grafico a barre (stipendio medio per mansione)
- `assets/ex07-line-chart.png` — Grafico lineare con dati normalizzati (CO₂, deforestazione, temperatura)

## How to review

1. Apri `Module03.xlsx`
2. Verifica ogni foglio Ex01–Ex07
3. Controlla le formule (nessun valore hardcoded)
4. Confronta i risultati con i grafici in `assets/`
5. Leggi `Module03.txt` per capire il ragionamento e i prompt usati

## Notes

Questo repository mostra il mio approccio a:
- scomposizione del problema
- traduzione di requisiti in logica formale
- separazione tra input, trasformazione e output
- verifica visiva dei risultati

Il focus è sulla **qualità del processo**, non sull’estetica.
